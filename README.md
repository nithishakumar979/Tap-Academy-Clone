## 🎯 Project Overview

This project is a responsive frontend clone of the TAP Academy website created using **HTML5, CSS3, and JavaScript**. The objective was to closely recreate the original website's layout, typography, spacing, color palette, and responsive behavior while improving frontend development skills.

---

## ✨ Features

- ✅ Responsive Navigation Bar
- ✅ Hero Section with CTA Buttons
- ✅ Placement Statistics Section
- ✅ Courses Section
- ✅ Student Success Stories
- ✅ Testimonials
- ✅ Contact Section with Google Map
- ✅ Footer
- ✅ Mobile Responsive Design
- ✅ Smooth Hover Animations
- ✅ Clean UI
